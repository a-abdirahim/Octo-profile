## A nicer look at github profiles!
![Screenshot](https://github.com/a-abdirahim/Octo-profile/blob/master/public/assets/profile-page.png?raw=true "My profile")

## Built with
    * Node-js
    * Express-js
    * Github Api
    * pug
    * css

# Getting started 
    1. Install dependencies 
        npm install 

    2. Fire up the backend 
        npm start
